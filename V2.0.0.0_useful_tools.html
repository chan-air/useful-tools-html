欢迎来到管理系统

这是一个带有可伸缩侧边栏的现代化界面设计

侧边栏功能说明

将鼠标移动到左侧的紫色边栏上，侧边栏会自动展开显示完整的菜单项。

移开鼠标后，侧边栏会自动收起，只显示图标，节省页面空间。

设计特点

1. 平滑动画： 使用 CSS transition 实现流畅的展开收起效果。

2. 渐变背景： 侧边栏采用紫色渐变设计，视觉效果更加现代。

3. 圆角设计： 右侧圆角让界面更加柔和美观。

4. 响应式布局： 适配不同屏幕尺寸，在移动设备上也能良好显示。

交互体验

每个菜单项都有悬停效果，点击时会有轻微的缩放反馈。

菜单文字会在侧边栏展开时淡入显示，收起时淡出隐藏。

主内容区域会根据侧边栏的展开状态自动调整边距。
